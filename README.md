# Coordinate

A Swift library for handling coordinates.

## Example

``` swift
let origin = Coordinate(
  long: 100.561371,
  lat: 13.822495
)

let target = Coordinate(
  long: 100.613660,
  lat: 13.749423
)

origin.distance(to: target) // 10.0km
```

// swift-tools-version: 4.2
import PackageDescription
